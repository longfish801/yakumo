        • Writable apply(String outKey, def out, String clmapKey, String combiKey, Map binds)

          適用工程を実行します。
          クロージャの第一引数は出力先キー、第二引数は出力先を渡します。
          デフォルトのバインド引数として engine（ConvertEngineインスタンス）を設定します。

          throws:
          IllegalStateException クロージャマップを参照できません。
          throws:
          IllegalStateException クロージャを参照できません。
          Parameters:
          outKey - 出力先キー
          out - 出力先（Writer, String, Fileのいずれか）
          clmapKey - clmapの識別キー
          combiKey - clmapのコンビキー
          binds - バインド変数マップ
          Returns:
          Writable

        • protected File getDefaultLogDir()

          デフォルトのログフォルダ返します。
          デフォルトのログフォルダとして、一時フォルダ（システム変数java.io.tmpdir）配下の yakumoサブフォルダを返します。
          ログフォルダは再作成します。処理失敗時は WARNログを、成功時は INFOログを出力します。

          Returns:
          デフォルトのログフォルダ（ログフォルダの再作成失敗時は null）
