        • Method Detail

          • Map<String, URL> find(String path, List<String> includePatterns, List<String> excludePatterns)

            リソース名に対応するフォルダ配下のすべてのリソースをパッケージルートおよびパッケージから返します。
            フォルダからの相対パスが同じリソースが存在した場合は、パッケージルート配下を優先します。
            リソース名に対応するリソースがまったく存在しない場合は空のマップを返します。
            リソース名に対応するリソースがファイルだった場合は、そのファイルのリソースのみ返します。
            ファイル名のパターンリストについてはクラスCollectFileVisitorにて利用します。

            Parameters:
            path - リソース名
            includePatterns - ファイル名の適合パターンリスト
            excludePatterns - ファイル名の除外パターンリスト
            Returns:
            指定したリソースからの相対パスと、リソースを読みこむためのURLとのマップ
