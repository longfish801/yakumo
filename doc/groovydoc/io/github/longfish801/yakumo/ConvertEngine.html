        • Writable apply(String outKey, def out, String templateKey, Map binds)

          テンプレートを適用します。
          テンプレートは以下の順番で取得を試みます。

          1. テンプレートキーと一致するテンプレート。
          2. デフォルトのテンプレートキー（default）と一致するテンプレート。
          throws:
          YmoConvertException 出力先が未対応のクラスです。
          throws:
          YmoConvertException 適用できるテンプレートがありません。
          Parameters:
          outKey - 出力先キー
          out - 出力先（Writer, String, Fileのいずれか）
          templateKey - テンプレートキー
          binds - バインド変数
          Returns:
          Writable

        • Map converts()

          一括変換します。
          テンプレートには、以下のバインド変数を設定します。

          bltxtMap
          変換元キーと BLtxtインスタンスとのマップ。
          clmap
          クロージャマップ。
          bltxtMap
          出力先キー。
          Returns:
          出力先キーと Writableとのマップ

        • protected File getDefaultLogDir()

          デフォルトのログフォルダ返します。
          デフォルトのログフォルダとして、一時フォルダ（システム変数java.io.tmpdir）配下の yakumoサブフォルダを返します。
          ログフォルダは再作成します。処理失敗時は WARNログを、成功時は INFOログを出力します。

          Returns:
          デフォルトのログフォルダ（ログフォルダの再作成失敗時は null）

        • io.github.longfish801.bltxt.BLtxt load(String sourceKey, def source)

          変換対象から BLtxtインスタンスを生成してメンバ変数のマップに保持します。
          まず変換対象を Washshスクリプトで整形します。
          整形結果を BLtxt文書として構造化します。

          Parameters:
          sourceKey - 変換元キー
          source - 変換対象（Reader, String, File, URLのいずれか）
          Returns:
          BLtxt
